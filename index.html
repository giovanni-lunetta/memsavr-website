Frequently Asked Questions

Is this safe? Do you see my photos?

It is 100% safe. MemSavr is a local desktop utility that runs entirely offline on your computer. Your photos are never uploaded, tracked, or seen by anyone but you.

Does this work on Windows or Intel Macs?

Apple Silicon Macs only (M1, M2, M3, M4). Intel Macs and Windows are not currently supported. Future versions may expand compatibility.

My download is 100GB+. Will this work?

Yes! We specifically built Cloud Mode for this. Unlike other tools that crash trying to unzip everything at once, MemSavr streams one file at a time. It handles massive libraries effortlessly.

What is Cloud Mode?

Cloud Mode processes your files in batches of 500. After each batch, it pauses so you can upload to Google Photos and delete the local files ‚Äî perfect if you have limited disk space.

Can I resume if I stop midway?

Absolutely! MemSavr automatically tracks your progress. If you close the app or restart your computer, it will pick up exactly where you left off.

I got a weird error that I can't fix. What should I do?

If you're experiencing an error, first try restarting the app. If the problem persists, export your support logs by going to Help ‚Üí Export Support Logs in the app's menu. This creates a ZIP file with diagnostic information. Then, email the ZIP file to along with a description of the issue. This helps us identify and fix the exact cause of your error quickly!
